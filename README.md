# MoonJuice

- Lua -> Moon
- Elixir -> Juice

MoonJuice is the custom scripting language built for TASBox.
This repository is a work-in-progress extraction of the language from the engine
(to allow its use in other projects, such as LSPs),
and rewrite in Rust (so it doesn't take 3 years to set up a monorepo with intra- and inter-repository dependencies).
